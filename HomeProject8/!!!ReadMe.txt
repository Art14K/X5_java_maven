Для работы программы необходим файл конфигурации config.conf.
При его отсутствии программа сама создаст файл с параметрами по умолчанию.
Доступно два режима работы программы - с файловой БД и с sql
Структура файла конфигурации:

mode:file - задаёт режим работы программы (файловая бд или sql)
file base name:base.txt - имя файловой базы
login:none - логин для подключения к sql - серверу
password:none - пароль для подключения к sql серверу
base:none - имя базы для подключения к sql серверу
server:none - адрес sql сервера
